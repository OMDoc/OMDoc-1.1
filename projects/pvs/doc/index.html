An OMDoc interface to PVS

In this project we develop interface code for the PVS theorem prover. The project code is available by anonymous CVS (path omdoc/projects/pvs), and can viewed here.

Authors and Contact

Michael Kohlhase (Carnegie Mellon University; primary author and contact) and Sam Owre (SRI International; PVS consulting)

Components

  • pp-omdoc.lisp: some pvs lisp code (based on the new release of PVS) that exports formulae in OMDoc. It works on the internal (parsed and disambiguated by PVS) version of the PVS data structures.
  • pvs.omdoc defines the OpenMath symbols of the PVS language constructs.
  • omdoc2pvs.xsl a style sheet for transforming OMDoc into PVS syntax.
  • you can find the results of exporting the PVS prelude here This is correct OMDoc (wrt. to the local version of omdoc.dtd, maybe needs update), and can partially be transformed back by the omdoc2pvs.xsl a style sheet. For the first 3n+k theories n,k>4, this can be parsed back by PVS (I have not checked that the theories are equivalent).
